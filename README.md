# 💍 Wedding Invitation Generator

Wedding Invitation Generator is a modern web application that allows users to create beautiful, interactive digital wedding invitations بسهولة (easily). It provides a simple way to design, customize, and share wedding details online.

🔗 Live Demo: https://weddinginvitation-generator.vercel.app/

---

## ✨ Features

- 💌 Generate digital wedding invitations instantly  
- 🖼️ Add couple photos  
- 📅 Display event details (date, time, venue)  
- 📍 Location integration (map / address)  
- 💬 Custom wishes or messages section  
- 🔗 Shareable invitation link  
- 📱 Fully responsive design  

---

## 🧠 Concept

This project is inspired by the shift from traditional paper invitations to **digital wedding invitations**, which are:

- More cost-effective  
- Faster to distribute  
- Easily customizable  
- Eco-friendly
